## 6. 缺失项与局限（诚实声明）

1. **仅模拟器**：所有量子结果来自 cqlib StatevectorSimulator，**非真实量子硬件**，不可描述为硬件性能。
2. **问题规模极小**：H2 仅 2 qubits，hardware-efficient ansatz 完全可表达，故 VQE 与精确解高度吻合——**这一良好结论不能外推到需要更多 qubit 的大分子**。
3. **无采样噪声**：使用 statevector 模拟而非 shot-based 测量，故无统计涨落；真实硬件/采样场景需额外考虑测量次数影响。
4. **SFC 未拆子组件**：`qccp-page-output/vqeH2/` 的 `index.vue` 为单文件（395 行），未像 Finance_QRC 那样拆分 components（功能完整，属可选优化）。
5. **未集成进 qccp-web 工程**：SFC 已按标准产出，但未执行 `npm run build` 接入真实 qccp-web（无该工程源码），集成步骤见 `INTEGRATE.md`。
