# Writing Principles

## Core Philosophy

> Writing a paper requires relentless polishing — repeatedly refining and asking whether readers can follow.
> The ability to self-assess whether your writing is clear is critical. You must recognize problems before you can fix them.

**Two keys to writing well:**
1. Clarify the writing plan before writing
2. Polish repeatedly — refine both the plan and the English sentences

---

## Paragraph Principles

### One Message Per Paragraph

> Each paragraph should convey only one message, expressed clearly. Do not mix multiple messages into one paragraph.

Each paragraph conveys exactly **one** point, idea, or message. If you find a paragraph covering multiple ideas, split it.

### Topic Sentence First

> The first sentence of a paragraph must immediately tell readers what it is about.

The first sentence of every paragraph tells the reader what this paragraph is about. Readers should never be confused about a paragraph's purpose after reading its first sentence.

---

## Writing Process

### Plan Before Writing

> Outline the writing plan first, refine each part, then write the English sentences. Pay attention to the flow between paragraphs and sentences.

1. **Outline**: List the writing plan for the entire section
2. **Refine**: Break each point into sub-points
3. **Draft**: Write English sentences following the refined plan
4. **Flow check**: Ensure logical flow between paragraphs and between sentences

### Reverse-Outlining

> Extract the writing plan from finished paragraphs and check whether the flow is smooth.

After writing a section:
1. Read each paragraph
2. Write down the main message of each paragraph in one sentence
3. Read the sequence of messages — does it flow logically?
4. If the flow breaks, reorganize paragraphs or add transitions

This technique catches structural problems that are invisible when reading paragraph by paragraph.

---

## Sentence-Level Principles

### Motivation Clarity

> Check that the motivation for every sentence is clear — readers should always know why they are reading it.

For every sentence, the reader should understand *why* they are reading it. If a sentence appears without clear motivation, add context or restructure.

### Sentence Flow

Ensure logical continuity between consecutive sentences:
- Each sentence should connect to the previous one
- Use transitional words/phrases when the connection is not obvious
- Avoid abrupt topic shifts within a paragraph

### Terminology Consistency

> Check that terminology is consistent — do not keep changing names for the same concept.

Pick one name for each concept and use it throughout the paper:
- **Bad**: "our network" / "the model" / "our framework" / "the system" (referring to the same thing)
- **Good**: "our network" consistently throughout

---

## Self-Assessment: Is the Writing Clear?

Read from the reader's perspective and check:

1. Does the paragraph have a clear topic?
2. Does the first sentence explain what this paragraph discusses?
3. Can readers understand every noun/concept in the sentences? Is it self-contained?
4. Is the logic between two consecutive sentences continuous?

> Important: Read from the reader's perspective. Check each paragraph's topic, opening sentence, concept clarity, and logical continuity.

## Section-Specific Tips

### For Introduction
- Use the "reverse then forward" thinking process
- Make sure the challenge naturally leads to the contribution
- Every contribution must have experimental support

### For Method
- Use the three-element system (design, motivation, advantages)
- One subsection per pipeline module
- Forward process description should be step-by-step

### For Experiments
- Table captions describe setup, not results
- Visual comparisons complement quantitative results
- Every ablation must be clearly interpretable

### For Abstract
- Write it last (Step 9)
- Answer the four pre-writing questions first
- Keep within word limits

---

## The Perfectionist Approach

> Strive for perfection:

1. **Adversarial writing**: Review your own paper as a reviewer would. Consider every question a reviewer might ask, and address them preemptively.
2. **Seek feedback**: Ask your advisor to review — the more feedback, the better.
3. **Multiple passes**: Each pass focuses on a different aspect (structure, clarity, grammar, figures, references).
